# RAW — A Guide to Which AI to Use in the Agentic Era (Ethan Mollick, One Useful Thing)
Source: https://www.oneusefulthing.org/p/a-guide-to-which-ai-to-use-in-the · 2026-02-18 · subtitle "It's not just chatbots anymore" · captured 2026-06-22 · channel: blog

- "Eighth of these guides since ChatGPT" but "a very large break with the past" because what "using AI" means has changed: until a few months ago it meant talking to a chatbot; now it's practical to use AI as an agent (assign a task, it does it using tools). So you now must consider three things: **Models, Apps, and Harnesses**.
- **Models** = the underlying AI brains. Big three: **GPT-5.2/5.3, Claude Opus 4.6, Gemini 3 Pro**. Determine how smart/how well it reasons/writes/codes/sees. "What the benchmarks measure."
- **Apps** = the products you use to talk to a model and get work done — websites (chatgpt.com, claude.ai, gemini.google.com), phone apps, plus coding tools (OpenAI Codex, Claude Code) and desktop tools (Claude Cowork).
- **Harnesses** = "what let the power of AI models do real work, like a horse harness." A system that lets the AI use tools, take actions, complete multi-step tasks on its own. Apps come with a harness. Claude.ai's harness = web search + code + instructions for spreadsheets/design. Claude Code's harness is more extensive: virtual computer, web browser, code terminal, strung together. **Manus** (recently acquired by Meta) = a standalone harness wrapping multiple models. **OpenClaw** (big recent news) = mostly a harness to use any model locally.
- Central point: the same model behaves very differently by harness. "Claude Opus 4.6 talking to you in a chat window is a very different experience from Claude Opus 4.6 operating inside Claude Code." So "which AI should I use?" now depends on what you're trying to do.
- **Models right now**: top models remarkably close, smarter & fewer errors than ever. Must pay ≥$20/mo for serious use. Free models are optimized for chat (fast, fun) not accuracy/capability — "when someone posts an example of an AI doing something stupid, it is because they are either using the free models or… have not selected a smarter model."
- **The single most important thing: pick the right model** (companies don't make it easy). ChatGPT default "GPT-5.2" is really "auto" mode (often a weaker model); he always manually picks **GPT-5.2 Thinking Extended** ($20) / **Thinking Heavy** (pricier) / GPT-5.2 Pro (strongest, higher tier). Gemini: pick **Gemini 3 Pro or Thinking** (Deep Think on Ultra). Claude: pick **Opus 4.6** + turn on "extended thinking" (Sonnet 4.6 also powerful, not quite as good).
- "For most people, the app and harness matter more than the model."
- **Chatbot interfaces** diverging. Bundled features: Gemini (nano banana image gen — "the best current AI image creation tool", Veo 3.1 video, Guided Learning, Deep Research); ChatGPT (Create Images, Study and Learn, Quizzes, Deep Research, Shopping Research); Claude (Deep Research; study mode via Projects). All can connect to your data (email, calendar, files) via connectors.
- On harnesses among chatbots: **OpenAI and Anthropic have clear leads over Google**. Claude.ai and ChatGPT can write/execute code, give files, do extensive research; Gemini's website is much less capable despite an equally good model (can't produce working spreadsheets/PowerPoints, no citations) — "I do expect Google will catch up here soon."
- GPT-5.2 Pro = a VERY smart model (helped derive a novel result in theoretical physics); most capable for complex statistical/analytical work; expensive plans only. Gemini 3 Deep Think very capable but same harness problem.
- **Other apps and harnesses** (where the most impressive work happens):
  - **Claude Code, OpenAI Codex, Google Antigravity** — most developed, aimed at coders (access to codebase, terminal, write/run/test code). Even non-coders can do a tremendous amount. Anecdote: asked Claude Code to make a paper-based LLM — 80 laid-out volumes of all of GPT-1's 117M weights + a math guide + book covers visualizing the weights + an elegant website hooked to Stripe + Lulu print-on-demand, tested and launched, "I never touched or looked at any code." Sold out the day he listed 20 (free PDFs at weights-press.netlify.app). "Coding harnesses remain risky for amateurs."
  - **Claude for Excel and PowerPoint** — specific harnesses inside apps. Claude for Excel "feels like a massive change in working with spreadsheets… a sort of junior analyst," results easy to check because they're in Excel. Google has lighter Sheets integration; OpenAI has no equivalent.
  - **Claude Cowork** — "something genuinely new… deserves its own category." Released by Anthropic in January; "essentially Claude Code for non-technical work." Runs on your desktop, works with local files and browser, but "much more secure than Claude Code" — runs in a VM with default-deny networking and hard isolation. You describe an outcome (organize expense reports, pull PDF data into a spreadsheet, draft a summary), Claude plans, breaks into subtasks, and executes on your computer. Built on the same agentic architecture as Claude Code and "itself largely built BY Claude Code in about two weeks." Neither OpenAI nor Google has a direct equivalent. Still a research preview ("eats through usage limits fast").
  - **NotebookLM** (Google) — make sense of a lot of information; add papers/YouTube/sites/files; query, turn into slides/mind maps/videos/AI podcasts (interruptible). Great for students/researchers.
  - **OpenClaw** — open-source AI agent that went viral late January; runs locally, connects to any model, you talk to it via WhatsApp/iMessage; browses web, manages files, sends email, runs commands; a "24/7 personal assistant." "A serious security risk" — "you almost definitely shouldn't use" it.
- **What to do now**: getting started → pick one of ChatGPT/Claude/Gemini, pay $20, select the advanced model; "invite AI to everything," upload a real document, give it a complex task as an RFP/SOP, push it. Comfortable already → try NotebookLM (free), then go deeper with Claude Code / Claude Cowork (Claude Desktop) + the PowerPoint/Excel plugins — "not as a demo, but with something you actually need done." "You aren't prompting, you are managing."
- Close: "The shift from chatbot to agent is the most important change in how people use AI since ChatGPT launched… an AI that does things is fundamentally more useful than an AI that says things."

Stake: Wharton/UPenn associate professor; Co-Intelligence author. Note heavy Anthropic favorability here (he describes Claude Cowork in detail, gets early access) — discount the product enthusiasm; weight the Models/Apps/Harnesses taxonomy and the "pick the right model" practitioner advice.
