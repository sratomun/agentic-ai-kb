Abstract
Tool-integrated reasoning (TIR) enables LLM
agents to solve tasks through planning, tool use,
and iterative revision, but outcome-only reinforcement learning in this setting suffers from
sparse, delayed rewards and weak step-level
credit assignment. In long-horizon TIR trajectories, an early irrecoverable mistake can determine success or failure, making it crucial to
localize the first irrecoverable step and leverage
it for fine-grained credit assignment. We propose Error-Localized Policy Optimization
(ELPO), which localizes the first irrecoverable step via binary-search rollout trees under
a fixed rollout budget, converts the resulting
tree into stable learning signals through hierarchical advantage attribution, and applies errorlocalized adaptive clipping to strengthen corrective updates on the critical step and its suffix.
Across TIR benchmarks in math, science QA,
and code execution, ELPO consistently outperforms strong Agentic RL baselines under
comparable sampling budgets, with additional
gains in Pass@K and Major@K scaling, rollout
ranking quality, and tool-call efficiency. Our
code will be publicly released soon.

current outcome reward based reinforcement techniques, represented by GRPO (Shao et al., 2024).

Related Work

2.1

Tool-integrated Reasoning Agents

Recent studies have explored enabling LLMs to
act as decision-making agents equipped with external tools such as search engines, calculators, code
interpreters, and APIs (Yao et al., 2022; Schick
et al., 2023). Early approaches primarily relied
on prompting strategies such as ReAct (Yao et al.,
2022) and Reflexion (Shinn et al., 2023) to interleave reasoning with tool calls, while later work
examined finetuning LLMs on tool-usage traces to
improve robustness in multi-step reasoning scenarios (Qin et al., 2023; Tang et al., 2023). This line
of work highlights both the promise and the difficulty of training reliable tool-augmented agents,
particularly when errors made early in the reasoning process propagate through subsequent steps.
2.2

Reinforcement Learning for LLM Agents

Reinforcement learning has emerged as a key technique for improving long-horizon reasoning in
LLMs. Group-based policy optimization methods
such as GRPO (Shao et al., 2024) and DAPO (Yu
et al., 2025a) avoid explicit value networks by normalizing rewards among multiple rollouts for the
same prompt, enabling large-scale RL training for
reasoning, search, and tool-use tasks. Building on
this foundation, early Agentic RL systems such as
ToRL (Li et al., 2025) and Search-R1 (Jin et al.,
2025) apply outcome-based RL to tool-augmented
reasoning, showing that verifiable terminal supervision can elicit effective tool use (e.g., code execution and web search). Subsequent work (Huang
et al., 2025; Feng et al., 2025; Lin and Xu, 2025)
improves learning efficiency and stability by introducing finer-grained reward design or advantage
shaping on top of outcome supervision.
Tree-structured rollouts have also become an active direction for extracting richer training signals.
Tree-GRPO (Ji et al., 2025) constructs a rollout
tree via stochastic expansion and leverages the tree
structure for fine-grained advantage estimation, but
its random branching can be sensitive to the initial
trajectory quality. More recent methods (Lu et al.,
2025; Dong et al., 2025) further use entropy-guided
expansion to steer branching and encourage more
diverse sampling under the same rollout budget.
Despite this progress, existing approaches

Tree Rollout Based on Binary Error
Localization (BEL)

In ELPO, we use a binary-search rollout tree to
pinpoint the first irrecoverable step in a failed trajectory. Concretely, given a failed trajectory, we
repeatedly select the middle step as an anchor step.
The anchor step and all preceding steps are fixed
as a prefix, and several completions are sampled
for the remaining suffix. If any suffix completion
succeeds, the prefix is recoverable and we search
the later half; otherwise it is irrecoverable and we
search the earlier half. We repeat this process until
the range collapses to a single step, which localizes the first irrecoverable step with only O(log K)
anchor probes rather than exhaustively testing every step. The main practical challenge is budget
control: binary localization requires extra suffix
completions, but the per-input rollout budget must
remain fixed. We address this with two complementary mechanisms.

Implementation Details

We implement ELPO on top of the VERL framework3 with feedback-token masking, excluding
tool-call feedback tokens from the loss so gradients are accumulated only over agent-generated
tokens. We conduct all training on two backbone
instruction-tuned LLMs, Qwen2.5-7B-Instruct4
and Qwen3-4B-Instruct5 . Following DemyAgent (Yu et al., 2025b), we first perform a cold-start
SFT for 5 epochs on the Open-AgentRL-SFT-3K
dataset6 to acquire basic tool-use patterns, and then
run RL training for 1 epoch on the Open-AgentRL30K dataset7 under the same data construction protocol. The GRPO KL regularization coefficient is
set to 0 for stability.
Our training uses a total batch size of 128 with a
PPO mini-batch size of 16, and a maximum context
length of 20K tokens. For every method, we enforce the same per-input rollout budget Ntotal = 16.

Conclusion

We study outcome-only Agentic RL for longhorizon TIR, where sparse terminal rewards lead
to diffuse credit assignment and obscure the earliest irrecoverable decision. We propose ErrorLocalized Policy Optimization (ELPO), which
uses binary-search tree rollouts to localize the first
irrecoverable step under a fixed rollout budget, and
turns this signal into stable updates via hierarchical
advantage attribution and error-localized adaptive
clipping. Experiments on tool-augmented benchmarks in math, science QA, and code execution
show that ELPO consistently outperforms strong
Agentic RL baselines under comparable sampling
budgets, with better Pass@K/Major@K scaling
and improved tool-call efficiency. Overall, ELPO
demonstrates that explicitly localizing and exploiting irrecoverable errors provides a practical path
to more targeted and reliable optimization for long8

Despite its effectiveness, ELPO has several limitations. First, the localization of the first irrecoverable step is an empirical result obtained under a
fixed rollout budget, current policy, and decoding
configuration, based on finite suffix sampling. In
low-success or highly stochastic settings, rare successes or false negatives may introduce bias or variance into the binary localization process, and the
identified step may shift as the policy or sampling
budget changes. Second, our evaluation focuses on
relatively deterministic tool environments such as
code execution, calculation, and closed-form question answering, and we have not systematically
tested ELPO in noisier or more interactive settings,
e.g., open-domain web search or GUI-based tools,
where tool feedback uncertainty may further affect
recoverability judgments. Finally, ELPO currently
targets a single earliest critical step for localized
updates, which may not fully capture failure modes
arising from multiple interacting errors or gradual
error accumulation.

This work improves reinforcement learning for TIR

credit assignment under a fixed rollout budget. We
follow standard academic ethical practices and use
only publicly available tools and benchmark environments (e.g., code execution and calculators),

access. Although stronger tool-use ability could be
misused in deployment, ELPO is a training technique rather than a mechanism for data collection
or deception; responsible use should include appropriate access control and auditing. We do not

the experiments reported in this paper.
