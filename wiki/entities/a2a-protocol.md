# A2A (Agent2Agent) Protocol

*Open protocol for peer agent coordination — discovery, negotiation, and task delegation between autonomous agents.*

## What it is
A2A is an emerging open protocol standardizing how agents interact with *each other*: task discovery, negotiation, and delegation between autonomous agents across systems and organizations. It complements [[mcp]] (agent-to-tool) by adding the agent-to-agent coordination layer — together the two protocols are framed as the interoperable substrate for a distributed 'agentic web'.

## Why it matters
Without A2A, every multi-agent deployment invents its own coordination wire — each proprietary integration is lock-in and a security surface. A2A standardization is the prerequisite for composable, interoperable agent services at enterprise scale.
