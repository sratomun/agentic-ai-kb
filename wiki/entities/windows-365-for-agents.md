# Windows 365 for Agents

*Microsoft's managed Cloud PC execution layer for computer-using agents — full Windows sessions governed by Entra ID, Intune, and Conditional Access.*

## What it is
Windows 365 for Agents (announced at Build 2026) gives agents a provisioned Cloud PC to complete tasks that APIs can't handle — UI-level actions, human-in-the-loop workflows, apps without reliable APIs. Microsoft handles session lifecycle and isolation; identity and policy run through Microsoft Entra ID, Intune, and Conditional Access.

## Why it matters
The governance model is the signal: agents get enterprise-grade identity and policy enforcement from day one. Microsoft is shipping the compliance infrastructure for agents before enterprise buyers can demand it — setting the bar other platforms will have to match.
