# ALFWorld

*A text+embodied benchmark aligning abstract instructions with household tasks — a staple for embodied and long-horizon agent evaluation.*

## What it is
ALFWorld aligns a text-based agent environment (ALFRED) with interactive 3D household tasks, letting agents practice abstract instruction following in a household setting. It appears in ~96 of 2026 H1 census papers.

## Why it matters
Household task completion is a widely used proxy for long-horizon agent reliability. ALFWorld's text+embodied framing makes it useful for both language-only and multimodal agent evaluation pipelines.
