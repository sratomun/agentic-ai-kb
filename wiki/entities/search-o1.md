# Search-o1

*Agentic search-enhanced reasoning — interleaving retrieval with long reasoning chains, an influential agentic-RAG method.*

## What it is
Search-o1 is an agentic method that interleaves web search and retrieval within long chain-of-thought reasoning, allowing a model to fetch evidence mid-reasoning rather than pre-loading context. It is a foundational agentic-RAG design.

## Why it matters
Search-o1 demonstrated that retrieval and reasoning don't have to be separate pipeline stages — folding search into the reasoning loop produces better answers on knowledge-intensive tasks. This pattern is now standard in deep-research and enterprise knowledge-work agents.
