# Agent Laboratory

*An LLM-agent framework that automates the full research workflow from literature review to experiments to writing.*

## What it is
Agent Laboratory is an LLM-agent framework that runs the scientific research workflow (literature review → experiments → writing) as an autonomous assistant, designed to accelerate scientific discovery.

## Why it matters
Autonomous science agents are a near-term enterprise bet — Lab R&D, drug discovery, and engineering all have high-value research loops. Agent Laboratory is the reference architecture for that class of agent.
