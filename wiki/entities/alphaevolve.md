# AlphaEvolve

*A DeepMind coding agent that evolves algorithms via LLM-guided search — a flagship autonomous scientific discovery system.*

## What it is
AlphaEvolve is a coding agent from Google DeepMind that uses LLM-guided evolutionary search to discover and improve algorithms for scientific and mathematical problems.

## Why it matters
AlphaEvolve shows that agents can produce net-new scientific results, not just automate existing work. It's the clearest current example of AI as a research accelerant rather than a productivity tool.
