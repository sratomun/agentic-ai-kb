# A-MEM

*An agentic memory framework using Zettelkasten-style interlinked notes — one of the most-cited 2025 agent-memory works.*

## What it is
A-MEM (Agentic Memory) is a memory framework for LLM agents that creates and updates Zettelkasten-style interlinked notes dynamically, giving agents structured long-term memory. Cited in 651 papers as of 2025.

## Why it matters
Memory is the missing link for reliable long-horizon agents. A-MEM's note-graph approach — rather than flat retrieval — is the architecture pattern most likely to survive into production agent memory systems.
