# MLE-bench

*A benchmark of real ML-engineering tasks (Kaggle-style competitions, data pipelines) for evaluating autonomous coding and research agents on end-to-end ML workflows.*

## What it is
MLE-bench draws tasks from real Kaggle competitions and ML engineering workflows — feature engineering, model selection, training, and evaluation — requiring agents to produce working ML solutions autonomously. Appears in ~22 of the 2026 H1 census papers as a standard for ML-automation agent evaluation.

## Why it matters
Enterprise AI teams automating model development and data workflows need agents that perform on MLE-bench — it's the test for whether a coding agent can do real data science, not just synthesize toy code samples.
