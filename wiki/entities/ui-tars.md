# UI-TARS

*A native GUI agent model for automated computer/UI interaction — the landmark 2025 computer-use system from ByteDance.*

## What it is
UI-TARS is a multimodal foundation model from ByteDance's Seed team trained natively for GUI agent tasks: perceiving screens and generating precise interaction actions. It set state-of-the-art results on OSWorld, ScreenSpot, and other computer-use benchmarks in early 2025.

## Why it matters
UI-TARS demonstrated that a purpose-built GUI model substantially outperforms general VLMs on computer-use tasks — the argument for training specialized agent models rather than prompting general ones. It's the reference architecture for enterprise computer-use agent development.
