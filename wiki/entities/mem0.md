# Mem0

*A production-oriented scalable long-term memory layer for AI agents — the reference implementation for deployable agent memory.*

## What it is
Mem0 is a memory framework for AI agents that manages long-term, session-persistent memory at production scale. It provides structured storage and retrieval of agent state, user preferences, and conversation history across sessions.

## Why it matters
Most agent memory solutions break in production — they're too slow, too lossy, or too expensive at scale. Mem0 is the most-cited production-grade solution (412 citations), making it the reference point for any team evaluating agent memory architecture.
