# AgenticCyOps

*An architectural security framework for enterprise multi-agent systems — maps attack vectors to tool orchestration and memory management, defines five defensive principles against NIST/ISO-27001/GDPR/EU AI Act.*

## What it is
AgenticCyOps traces documented multi-agent attack vectors back to two root integration surfaces — tool orchestration and memory management — and defines five defensive principles mapped to NIST, ISO-27001, GDPR, and the EU AI Act. Demonstrated on a SOC workflow over [[mcp]].

## Why it matters
The compliance mapping is the key contribution: it translates multi-agent security risks into terms enterprise CISOs and GRC teams already speak. That bridge — from agent attack vector to regulatory standard — is what makes agentic security reportable to a board.
