# Cloud Security Alliance

*Industry body whose [[maestro|MAESTRO]] framework is the first threat-modeling method built specifically for agentic AI.*

## What it is
The Cloud Security Alliance, a nonprofit that sets best practices for cloud and, increasingly, AI security. Its agentic work includes [[maestro|MAESTRO]] (Multi-Agent Environment, Security, Threat, Risk, and Outcome) — a seven-layer threat-modeling framework — plus an effort to build a NIST AI RMF **Agentic Profile**.

## Why it matters
CSA is the *threat-modeling* anchor of the [[security-standards|standards layer]]: [[maestro|MAESTRO]] is the only framework purpose-built to reason about *agentic* systems layer by layer (foundation model → data → frameworks → infra → eval → orchestration → ecosystem), which maps cleanly onto the autonomy spectrum. Where OWASP gives you a list and ATLAS gives you techniques, MAESTRO gives you a structured way to find where a specific multi-agent deployment breaks.
