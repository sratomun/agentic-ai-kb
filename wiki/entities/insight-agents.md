# Insight Agents (IA)

*A production multi-agent data-insight system for Amazon sellers — 90% question accuracy at P90 <15s latency.*

## What it is
Insight Agents (IA) is a production conversational multi-agent data-insight system deployed for US Amazon sellers (Bai et al., SIGIR '25). A plan-and-execute manager routes over specialist workers, achieving 89.5% question-level accuracy (all of Relevance/Correctness/Completeness > 0.8) at P90 13.56s.

## Why it matters
Insight Agents is one of the few published case studies of a production multi-agent data system with rigorous latency and accuracy metrics. Its Relevance/Correctness/Completeness eval decomposition is a transferable template for enterprise data-agent QA.
