# τ-Trait (tau-trait)

*A customer-service benchmark stress-testing tool-using agents against high-fidelity human trait simulations — impatience, frustration — in a single-control setting.*

## What it is
τ-Trait (He et al., 2026 — "Impatient users confuse AI agents") tests whether user behavior, not just task complexity, degrades agent reliability. It pairs the same structured tool-use format as [[tau2-bench]] with realistic human persona simulations; LedgerAgent uses its Telehealth domain as one of four evaluation domains.

## Why it matters
Shows that user volatility (impatience, frustration) degrades agent performance even when the task is well-defined — a practical problem for any enterprise deploying agents in customer-facing roles where human behavior is unpredictable.
