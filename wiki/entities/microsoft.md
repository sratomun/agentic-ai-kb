# Microsoft

*Enterprise software and cloud platform vendor shipping the governance and execution infrastructure for enterprise agent deployment.*

## What it is
At Build 2026, Microsoft announced a stack of agent infrastructure: Windows Development Skills (GA), Intelligent Terminal 0.1, Microsoft Execution Containers (MXC) for agent sandboxing (OpenAI-endorsed), [[windows-365-for-agents|Windows 365 for Agents]] (Cloud PCs for computer-using agents governed by Entra/Intune/Conditional Access), Aion 1.0 Plan (14B on-device model), and forward look at Project Solara.

## Why it matters
