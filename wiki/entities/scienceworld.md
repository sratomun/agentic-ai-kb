# ScienceWorld

*An interactive text environment where agents run virtual scientific experiments — tests hypothesis-driven planning and multi-step reasoning under novel constraints.*

## What it is
ScienceWorld places agents in a simulated lab where they must design and execute experiments to answer scientific questions (biology, chemistry, physics) using available tools and objects in the environment. Appears in ~22 of the 2026 H1 census papers as a standard for embodied scientific reasoning.

## Why it matters
ScienceWorld stress-tests planning depth under novel constraints where rote tool-calling fails — a probe for whether agents can reason under uncertainty rather than pattern-match to known API sequences. Less directly enterprise-relevant but a useful signal for general-purpose reasoning quality.
