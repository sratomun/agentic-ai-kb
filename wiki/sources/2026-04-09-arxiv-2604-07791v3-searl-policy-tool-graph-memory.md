# SEARL: Joint Optimization of Policy and Tool Graph Memory for Self-Evolving Agents

**Why it matters:** Self-evolving agents for resource-constrained settings — couples policy learning with a structured tool-graph memory to densify sparse outcome rewards. Counters the 'only works with giant models' critique.

## Takeaways
- SEARL builds a structured experience memory integrating planning with execution, giving a state abstraction that generalizes across analogous contexts (e.g. tool reuse).
- Densifies sparse outcome-based RLVR rewards by exploiting inter-trajectory correlations.
- Targets deployment where large multi-agent frameworks are impractical; evaluated on knowledge reasoning and math.
