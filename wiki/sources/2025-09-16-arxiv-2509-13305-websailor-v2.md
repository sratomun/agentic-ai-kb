# WebSailor-V2: Bridging the Chasm to Proprietary Agents via Synthetic Data and Scalable Reinforcement Learning

**arXiv [2509.13305](https://arxiv.org/abs/2509.13305)** · 2025-09-16 · Kuan Li, Zhongwang Zhang, Huifeng Yin, et al. (Tongyi Lab, Alibaba Group)

**Significance.** Shows synthetic data + scalable agentic RL can close the open-vs-proprietary gap for deep-research web agents — a 30B MoE matching or beating 671B proprietary systems. A 2025 data-centric case study built on Qwen.

## Abstract
The development of autonomous agents capable of leveraging tools like search engines and web browsers to perform "Deep Research" is recognized as a critical milestone in AI. However, a significant performance gap persists between open-source agents and powerful proprietary systems. We present WebSailor-V2, an open-source web agent that bridges this chasm through two key innovations: a scalable synthetic data generation pipeline (SailorFog-QA-V2) and a robust, dual-environment reinforcement learning framework. Built on a Qwen3-30B-A3B model and the simple ReAct paradigm, WebSailor-V2 achieves state-of-the-art results among open-source agents and is competitive with leading proprietary systems across challenging Deep Research benchmarks.

## From the paper (full-text)
**Contribution / method.** Built on the **Qwen3-30B-A3B** Mixture-of-Experts model (128k context) using the ReAct framework. Data: **SailorFog-QA-V2** constructs *dense* knowledge graphs (with cyclic structures, not tree-like) from seed entities and live web tools, extracts non-isomorphic connected subgraphs via random walk (verified with the Weisfeiler-Leman algorithm), and generates QA with a wider array of uncertainty definitions beyond simple obfuscation. Training is SFT cold-start (entirely synthetic trajectories via rejection sampling) then agentic RL in a dual environment: a high-fidelity offline Wikipedia simulator for fast iteration, plus a fault-tolerant unified tool-execution interface for real web APIs. The RL algorithm is a tailored GRPO variant (strictly on-policy, token-level loss, leave-one-out advantage, conservative negative-sample handling), with a symbiotic data-policy feedback loop curating data in real time.

**Results.** WebSailor-V2-30B-A3B sets open-source state of the art and beats much larger proprietary models: BrowseComp-EN 35.3 (vs. 671B DeepSeek-V3.1 at 30.0); BrowseComp-ZH 44.1; Humanity's Last Exam (HLE) 30.6 — a new SOTA, exceeding DeepSeek-V3.1 (29.8) and OpenAI-o3 (20.2); xbench-DeepSearch 73.7; GAIA 74.1; DeepResearch Bench 48.9 (second only to Gemini-2.5-pro-DeepResearch at 49.7). SFT alone already scored 24.4 on BrowseComp-EN and 23.9 on HLE. Scaling context 32k→128k improved accuracy (~90% of solved instances finished within 64k); at a 32k limit the 30B MoE hit ~16% on BrowseComp-EN, beating the larger 72B-dense WebSailor-V1.

**Takeaway.** At this stage of agentic RL, the quality/distribution of synthetic data and the stability of the training environment matter more than the specific RL algorithm — enough to let a 30B open model rival 671B proprietary agents.
