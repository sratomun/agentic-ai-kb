# Beyond Static Leaderboards: Predictive Validity for the Evaluation of LLM Agents

**Why it matters:** The sharpest 2026 critique of agent leaderboards: aggregate-score rankings don't transfer out-of-distribution. Proposes ranking by predictive validity instead. Essential skeptic lens for reading any agent benchmark.

## Takeaways
- Aggregates 14 parallel studies on one MCP-based industrial-agent benchmark + 7 prior benchmarks — no single benchmark covers more than 4–5 deployment dimensions.
- Aggregate-score leaderboards systematically underspecify deployed-agent evaluation; rankings don't transfer OOD (rank instability confirmed by public-to-hidden retrospectives).
- Proposes ranking configurations by predictive validity (in-sample↔out-of-sample rank correlation) + a 12-tier measurement apparatus; offers three falsifiable OOD criteria.
