# From Storage to Experience: A Survey on the Evolution of LLM Agent Memory Mechanisms

**SURVEY** · arXiv [2605.06716v1](https://arxiv.org/abs/2605.06716v1) · 2026-05-07 · Jinghao Luo, Yuchen Tian, Chuxue Cao, Ziyang Luo et al.

**Why it matters:** A clean mental model for where agent memory is heading — Storage→Reflection→Experience is a useful ladder for evaluating any memory system.

## What it surveys
The evolution of LLM-agent memory mechanisms, bridging the OS-engineering vs cognitive-science divide in the literature.

## Framework / taxonomy
Proposes a three-stage evolutionary framework: Storage (trajectory preservation) → Reflection (trajectory refinement) → Experience (trajectory abstraction), driven by long-range consistency, dynamic environments, and continual learning.

## Notable points
- Frontier 'Experience' stage centers on proactive exploration and cross-trajectory abstraction.
- Gives design principles and a roadmap rather than just a catalog.
- Reframes memory as the architectural cornerstone of agents.
