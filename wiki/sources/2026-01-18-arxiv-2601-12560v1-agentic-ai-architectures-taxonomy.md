# Agentic Artificial Intelligence (AI): Architectures, Taxonomies, and Evaluation of Large Language Model Agents

**Why it matters:** A field-map survey of the agentic-AI stack — the perceive/brain/plan/act/tool/collaborate decomposition is a clean shared vocabulary, and it names the move from fixed API calls to open standards (MCP, native computer use).

## Takeaways
- Proposes a unified taxonomy: Perception, Brain, Planning, Action, Tool Use, Collaboration — single-loop agents through hierarchical multi-agent systems.
- Frames the shift from linear reasoning to native inference-time reasoning models, and from fixed APIs to open standards ([[mcp]], native computer use).
- Names the recurring failure set: hallucination-in-action, infinite loops, prompt injection.
