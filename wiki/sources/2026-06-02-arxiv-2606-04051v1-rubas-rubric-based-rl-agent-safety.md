# RUBAS: Rubric-Based Reinforcement Learning for Agent Safety

**Why it matters:** Moves agent safety from coarse refusal signals to fine-grained rubric rewards over whole trajectories (tool-use / argument / response safety + helpfulness) — optimizing safe tool use without killing utility.

## Takeaways
- [[rubas|RUBAS]] decomposes agent behavior into four rubric dimensions: tool-use safety, argument safety, response safety, helpfulness.
- Structured rubrics give interpretable per-trajectory rewards for RL, balancing safety against task completion.
- Improves safety over standard alignment baselines, cuts tool-grounded hallucinations, keeps competitive utility.
