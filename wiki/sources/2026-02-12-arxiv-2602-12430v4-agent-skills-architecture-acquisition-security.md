# Agent Skills for Large Language Models: Architecture, Acquisition, Security, and the Path Forward

**Why it matters:** Companion survey to the Agentic-Skills SoK, anchored on the SKILL.md spec and progressive disclosure; reports 26.1% of community skills contain vulnerabilities and proposes a four-tier trust/governance model.

## Takeaways
- Four axes: architectural foundations (SKILL.md, progressive context loading, skills vs [[mcp]]), acquisition (RL with skill libraries, autonomous discovery/SEAgent, compositional synthesis), deployment (CUA stack, GUI grounding, OSWorld/SWE-bench), security.
- Empirical: 26.1% of community-contributed skills contain vulnerabilities → proposes a Skill Trust & Lifecycle Governance Framework (four-tier, gate-based permissions mapping provenance to capability).
- Positions the skill abstraction layer as distinct from generic tool-use surveys.
