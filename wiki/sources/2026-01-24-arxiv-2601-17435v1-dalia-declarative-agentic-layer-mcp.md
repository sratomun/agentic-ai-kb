# Towards a Declarative Agentic Layer for Intelligent Agents in MCP-Based Server Ecosystems

**Why it matters:** Argues agent unreliability is architectural, not a model limit — DALIA enforces a declarative discovery/planning/execution split so plans are grounded only in declared operations.

## Takeaways
- DALIA: a model-independent declarative layer that formalizes executable capabilities and exposes them via a discovery protocol.
- Constructs deterministic task graphs grounded exclusively in declared operations — separating discovery, planning, execution to constrain agents to a verifiable space.
- Targets hallucinated actions, unexecutable plans, and brittle coordination in [[mcp]]-based server ecosystems.
