# SoK: Agentic Retrieval-Augmented Generation (RAG): Taxonomy, Architectures, Evaluation, and Research Directions

**Why it matters:** The first systematization of Agentic RAG as a sequential decision process — names the reliability risks (compounding hallucination, memory poisoning, retrieval misalignment) that bite once retrieval runs in an autonomous loop.

## Takeaways
- Formalizes agentic retrieval-generation loops as finite-horizon POMDPs with explicit control policies and state transitions.
- Taxonomy by planning mechanism, retrieval orchestration, memory paradigm, and tool-invocation behavior.
- Flags systemic risks of autonomous loops: compounding hallucination propagation, memory poisoning, retrieval misalignment, cascading tool-execution vulnerabilities.
