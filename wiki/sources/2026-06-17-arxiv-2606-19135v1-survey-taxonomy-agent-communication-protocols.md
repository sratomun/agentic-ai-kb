# A Technical Taxonomy of LLM Agent Communication Protocols

**SURVEY** · arXiv [2606.19135v1](https://arxiv.org/abs/2606.19135v1) · 2026-06-17 · Linus Sander, Habtom Kahsay Gidey, Alexander Lenz, Alois Knoll

**Why it matters:** The reference map for the protocol wars — directly informs which agent-comm standards to bet on and why a layered stack (not one winner) is likely.

## What it surveys
The fragmented landscape of LLM agent communication protocols (nine actively-maintained open-source protocols with real adoption).

## Framework / taxonomy
A five-dimension technical taxonomy: counterparty, payload, interaction state, discovery mechanism, schema flexibility — derived via a rigorous iterative method.

## Notable points
- All sampled A2A protocols combine hybrid payloads with session-state persistence; decentralized discovery remains rare; schema flexibility is trending up.
- Predicts short-term convergence pressure toward protocols unifying agent-to-agent and agent-to-tool/data comms.
- Long-term: no single protocol wins; the field evolves toward a federated, layered protocol stack.
