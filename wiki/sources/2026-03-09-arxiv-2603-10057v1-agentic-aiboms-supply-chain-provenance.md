# SBOMs into Agentic AIBOMs: Schema Extensions, Agentic Orchestration, and Reproducibility Evaluation

**Why it matters:** Extends SBOMs into active, agent-generated AIBOMs — runtime provenance for AI supply chains using an MCP/A2A multi-agent architecture and CSAF/VEX semantics. Bridges agent tooling with compliance/standards.

## Takeaways
- [[aibom|Agentic AIBOMs]]: extend static SBOMs into active provenance artefacts via policy-constrained multi-agent reasoning.
- Three agents — environment reconstruction ([[mcp]]), runtime drift monitoring ([[a2a-protocol|A2A]]), policy-aware vuln/VEX reasoning (AGNTCY) — emit structured VEX exploitability assertions under CSAF v2.0 / ISO-IEC 20153:2025.
- Standards-aligned schema extensions to CycloneDX/SPDX; improves runtime dependency capture and reproducibility at low overhead.
